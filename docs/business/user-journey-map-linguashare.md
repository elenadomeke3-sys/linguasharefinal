# User Journey Map — LinguaShare

**Data:** 16 marca 2026

---

## Cel Dokumentu

Celem tego dokumentu jest zmapowanie pełnej ścieżki użytkownika od pierwszego kontaktu z platformą do długoterminowego użytkowania (retention), wraz z identyfikacją kluczowych momentów i możliwości.

---

## 1. Fazy User Journey

| Faza | Opis | Cel |
|------|------|-----|
| **Awareness** | Użytkownik dowiaduje się o platformie | Zainteresowanie |
| **Consideration** | Użytkownik bada platformę | Signup |
| **Activation** | Użytkownik po raz pierwszy korzysta | First value |
| **Retention** | Użytkownik wraca regularnie | Nawyk |
| **Referral** | Użytkownik poleca innych | Wzrost |

---

## 2. Awareness Phase

### 2.1 Touchpoints

| Kanał | Opis |
|-------|------|
| **FB Groups** | "Szukam materiałów do nauki angielskiego" - odpowiedź na post |
| **Google Search** | "materiały do angielskiego PDF" - SEO |
| **Content** | Blog post "5 sposobów na naukę czasów" |
| **Referral** | Polecenie od znajomego nauczyciela |

### 2.2 Actions

```
Użytkownik widzi reklamę/post/artykuł
↓
 Czyta nagłówek: "Materiały do nauki języków w jednym miejscu"
↓
 Czyta krótki opis: "Oszczędź 2-5h tygodniowo na szukaniu materiałów"
↓
 Klika w link (CTA)
```

### 2.3 Emotions

- Curiosity 🟡
- Skeptical 🟡
- Hopeful 🟢

### 2.4 Opportunities

- Clear value proposition
- Social proof (testimonials)
- Preview of materials

## 3. Consideration Phase

### 3.1 Touchpoints

| Kanał | Opis |
|-------|------|
| **Landing page** | Główna strona z ofertą |
| **Pricing page** | Cennik |
| **Blog** | Artykuły o nauczaniu |
| **Social proof** | Testimonials, numbers |

### 3.2 Actions

```
Użytkownik wylądował na landing page
↓
 Przegląda hero section
↓
 Czyta features: "Szukaj, pobieraj, oceniaj"
↓
 Przegląda cennik: Free / Premium
↓
 Czyta testimonials: "Oszczędzam 3h tygodniowo!"
↓
 KLIKA "ZACZNIJ ZA DARMO"
```

### 3.3 User Flow

```
Landing Page
    │
    ▼
[Scrolls to features]
    │
    ▼
[Checks pricing]
    │
    ▼
[Reads testimonials]
    │
    ▼
[Click CTA] → Signup Flow
```

### 3.4 Emotions

- Intrigued 🟢
- Evaluating 🟡
- Want to try 🟢

### 3.5 Pain Points

- Nie wiem czy to działa
- Za dużo tekstu
- Nie widzę materiałów

### 3.6 Opportunities

- ✅ Show, don't tell (preview materials)
- ✅ Clear pricing
- ✅ Easy signup

## 4. Activation Phase

### 4.1 Touchpoints

| Kanał | Opis |
|-------|------|
| **Signup form** | Rejestracja |
| **Onboarding** | Pierwsze kroki |
| **Search** | Wyszukiwanie materiałów |
| **First download** | Pierwsze pobranie |

### 4.2 Actions

```
KROK 1: Signup
[Enter email]
[Create password]
[Optional: Connect Google]
[Email verification]

KROK 2: Onboarding (optional bypass)
"Co chcesz robić?"
- Szukać materiałów (default)
- Dodawać materiały
[Skip for now]

KROK 3: Browse
[Wylądował na stronie głównej]
[Widzi katalog materiałów]
[Przegląda karty materiałów]

KROK 4: Search
[Wpisuje: "angielski B1 gramatyka"]
[Widzi wyniki]

KROK 5: First Download
[Kliknij materiał]
[Zobacz detail]
[Kliknij "Pobierz"]
[Dostępne pobrania: 2/3]
[File downloads]
✅ PIERWSZY VALUE MOMENT
```

### 4.3 Key Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| Signup completion rate | > 70% | Analytics |
| Time to first download | < 5 min | Analytics |
| Activation rate | > 50% | % users who download |

### 4.4 Emotions

- Excited 🟢
- Slightly confused 🟡 (if onboarding too long)
- Satisfied 🟢 (after download)

### 4.5 Pain Points

- Zbyt długi onboarding
- Nie widzę materiałów bez logowania
- Nie wiem jak szukać

### 4.6 Opportunities

- ✅ Skip onboarding
- ✅ Preview materials without login
- ✅ Guided search

## 5. Retention Phase

### 5.1 Touchpoints

| Kanał | Opis |
|-------|------|
| **Dashboard** | Moje konto, statystyki |
| **Email** | Powiadomienia, newsletter |
| **Collections** | Własne kolekcje |
| **Profile** | Moje materiały |

### 5.2 User Journeys (Regular User)

```
SCENARIUSZ A: Powrót po 1 tygodniu
[Loguje się]
[Przegląda nowe materiały]
[Pobiera 2 materiały]
[Zostawia ocenę]
[Log out]

SCENARIUSZ B: Powrót po 2 tygodniach
[Loguje się]
[Sprawdza swoje kolekcje]
[Dodaje nowy materiał]
[Włącza powiadomienia]
[Log out]

SCENARIUSZ C: Chce Upgrade
[Loguje się]
[Próbował pobrać - limit 3]
[Widzi banner Premium]
[Kliknij "Zobacz Premium"]
[Przegląda cennik]
[Kliknij "Kup teraz"]
[Stripe Checkout]
[Płatność]
✅ PREMIUM USER
```

### 5.3 Key Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| DAU/MAU | > 20% | Analytics |
| Week 1 retention | > 60% | Cohort analysis |
| Week 4 retention | > 40% | Cohort analysis |
| Downloads per user | > 3/mies | Analytics |

### 5.4 Emotions

- Habit forming 🟢
- Valued 🟢
- Connected 🟡

### 5.5 Pain Points

- Nie widzę wartości (low retention)
- Za dużo spamu email
- Kiepska jakość materiałów

### 5.6 Opportunities

- ✅ New materials notifications
- ✅ Personalized recommendations
- ✅ Engagement emails (not too many)

## 7. Churn Journey

### 7.1 Why Users Leave

| Reason | % of churned |
|--------|--------------|
| Nie widzę wartości | 40% |
| Za drogie | 20% |
| Kiepska jakość materiałów | 15% |
| Zbyt skomplikowane | 10% |
| Konkurencja lepsza | 10% |
| Inne | 5% |

### 7.2 Churn Prevention

```
[User hasn't logged in for 14 days]
    │
    ▼
[Email: "Miss you! 5 new materials for [language]"]
    │
    ▼
[User returns?]
    │
    ├── YES → Welcome back flow
    │
    └── NO → [28 days]
              │
              ▼
[Email: "Last chance! Premium for 50%"]
    │
    ▼
[User returns?]
    │
    ├── YES → Win-back flow
    │
    └── NO → Churned
```

## 11. Checklist User Journey

- [ ] Landing page with clear CTA
- [ ] Easy signup (< 3 fields)
- [ ] Skip onboarding option
- [ ] First download < 5 min
- [ ] Download celebration (confetti/success message)
- [ ] Email welcome series
- [ ] New materials notifications
- [ ] Inactive user email sequence
- [ ] Referral program
- [ ] NPS survey

---

## 12. Wnioski

**Kluczowe zasady User Journey:**

1. **Quick time-to-value:** Pierwszy download w < 5 minut
2. **Minimal friction:** Signup w < 1 minutę, bez zbędnych kroków
3. **Celebrate wins:** Po pierwszym pobraniu - pozytywny feedback
4. **Stay connected:** Regularne email, ale nie spam
5. **Win-back:** Nie pozwól zapomnieć - reminder emails
