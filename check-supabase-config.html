📋 Instrukcje sprawdzania w Dashboard Supabase

  1. Authentication → Email Templates:
    • Sprawdź czy szablon "Reset Password" istnieje
    • Kliknij "Reset Password" → zobacz podgląd emaila
  2. Settings → Authentication:
    • Site URL: powinno być https://linguashare.vercel.app
    • Redirect URLs: powinno zawierać:
      • https://linguashare.vercel.app/reset-password
      • https://linguashare.vercel.app/auth
  3. Settings → Email:
    • Sprawdź czy jest skonfigurowany provider (domyślny Supabase lub custom SMTP)
    • Jeśli używasz darmowej wersji, domena musi być potwierdzona
